Each day there are a number of repetetive tasks to do when I start my machine up, like starting the VPN, mounting drives, starting the same set of programs every day. I got bored pressing the same buttons over and over again, so wrote a shell script to start up the apps and apply settings as requried for work and or home. The lsetup.sh utility written by me takes a json config file which allows you to define the instructions to run as part of a group, so for example running ./lsetup.sh work apps will start up all your work apps as required, in the current workspace. I typically switch to another workspace and run ./lsetup.sh home apps to start my home applications.
